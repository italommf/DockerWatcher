# Vídeo de Background

Para usar um vídeo como background:

1. Coloque seu arquivo de vídeo na pasta `Frontend/public/`
2. Nomeie o arquivo como `background.mp4`
3. O vídeo será usado automaticamente como background

**Formato recomendado:**
- Formato: MP4 (H.264)
- Resolução: 1920x1080 ou maior
- Duração: Curta (5-30 segundos, será em loop)
- Tamanho: Preferencialmente < 10MB para melhor performance

**Notas:**
- O vídeo toca automaticamente em loop
- Está mudo por padrão
- Se o vídeo não for encontrado, um degradê animado será exibido como fallback
- O vídeo fica fixo atrás de todos os componentes
